# MVP Launch Playbook

## Purpose

Launch the MVP into a focused learning loop.

## When to Use

Use this playbook when the active Growth role needs a sequence of execution.

## Sequence

1. Read positioning
2. Draft landing page
3. Choose channels
4. Plan launch
5. Capture results

## Outputs

- Decision or action summary
- Updated source-of-truth files when requested
- Next recommended LeanOS command

## Navigation

Start from `../README.md`, choose a role in `../roles/`, load required skills in `../skills/`, then use this playbook.

# Navigation Chain

`AGENT.md -> Department AGENT.md -> Department README or Workflow -> Area AGENT.md/README.md -> Role -> Skills -> Playbook -> Output`

Root AGENT.md chooses the owning department. Department AGENT.md chooses a department workflow or active area. Area AGENT.md files, when present, choose the specialist role before skills and playbooks are loaded. Area README files remain the directory map. Outputs should update source-of-truth files only when requested or clearly needed.

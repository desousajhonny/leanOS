# Release Manager

## Purpose

Prepare release readiness across CI/CD, environments, deployment, rollback and post-release checks.

## Use When

- a release needs readiness review
- a PR is close to merge
- post-merge continuation needs operational checks
- rollback or monitoring risk exists

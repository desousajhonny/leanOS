# MVP Delivery

## Purpose

Turn product strategy into executable MVP scope.

## Area

`operations/core`

## Sequence

1. Read product strategy
2. Define MVP scope
3. Write user stories
4. Define acceptance criteria
5. Confirm non-goals

## Outputs

- Decision or action summary
- Updated source-of-truth files when requested
- Next recommended LeanOS command

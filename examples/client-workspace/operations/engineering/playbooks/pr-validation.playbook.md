# PR Validation

## Purpose

Validate implementation before merge.

## Area

`operations/engineering`

## Sequence

1. Read PR context
2. Check scope
3. Review tests
4. Check coherence
5. Recommend merge or changes

## Outputs

- Decision or action summary
- Updated source-of-truth files when requested
- Next recommended LeanOS command
